Caesar

Caesar Cipher is a method of encryption which uses shifts in the alphabet to change a string of characters into a hidden message. The method takes the alphabet and shifts the letters x places right to create the encrypted test.

For example, a Caesar shift of 3 with the string "da" gives the encrypted text "gd" as illustrated below.
